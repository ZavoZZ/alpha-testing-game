# Kilo Code Rules Index

This file lists all workspace rules for this project.

## Rules

| Rule | File | Purpose |
|------|------|---------|
| Codebase Indexing | `codebase-indexing.md` | Enforce automatic indexing usage |
| Project Context | `project-context.md` | Project metadata and standards |
| Code Conventions | `code-conventions.md` | Code style and patterns |

## Usage

These rules are automatically loaded by Kilo Code for all conversations in this workspace.

## Adding New Rules

1. Create new `.md` file in this directory
2. Follow the format:
   ```markdown
   # Rule Name
   
   Description...
   
   ## Guidelines
   - Guideline 1
   - Guideline 2
   ```
3. Rules are loaded automatically
